docker-gitbook
==============

Environment is building with Docker.

# Usage

Install [Docker] [1]

    # Clone this repo

        git clone https://github.com/buggers/docker-gitbook.git
        cd docker-gitbook

    # Building Image

        make image

    # Running gitbook build

        make build

    # Running gitbook init

        make init


[1]:https://docs.docker.com/
